Retail-Dashboard
Project Objective
A Retail dashboard is a performance and activity tracking tool focusing on relevant KPIs and Metrics for the retail and trade sectors and real-time insights into the sales performance of individual stores with retail analysis. Retails Sale Dashboard tracking the month wise product purchased. Excel file used and more than 20,000 rows and 15+ column.
